ctrlp-cdnjs
===========

CtrlP extension for `cdnjs.com`.

Usage
-----

```vim
:CtrlPCdnJs
```

If may add map into your vimrc like below: >

```vim
noremap <leader>js :<c-u>CtrlPCdnJs<cr>
```

Requirements
------------

- [webapi](https://github.com/mattn/webapi-vim)
